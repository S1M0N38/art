# Francesco Luchino — Opere

Portfolio site for Italian painter Francesco Luchino. ~676 oil paintings displayed in a dark-room masonry gallery with full-screen lightbox.

**Live site:** [s1m0n38.github.io/art](https://s1m0n38.github.io/art/)

## Prerequisites

- [Node.js](https://nodejs.org/) ≥ 22

## Getting started

```bash
npm install
```

## Development

Start the local dev server with hot reload:

```bash
npm run dev
```

Open [localhost:4321/art](http://localhost:4321/art/).

## Production build

Generate a static build in `dist/`:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

Open [localhost:4321/art](http://localhost:4321/art/).

## Deployment

Push to `main` to trigger the GitHub Actions workflow (`.github/workflows/deploy.yml`), which builds and deploys to GitHub Pages automatically.
